=== Exposure: Space — external cosmic objects ===

Drop your own objects here (config/exposure_space/objects/) as .json files —
WITHOUT rebuilding the mod. File name = object id (my_comet.json -> exposure_space:my_comet).
After editing run /reload in game — the registry re-reads and re-syncs to clients.

See objects/example_comet.json.txt for a ready-made example — rename to .json and edit.
